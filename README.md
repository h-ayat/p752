# p752
Basic TUI engine for scala-native.

## Overview
This project consists of three modules:
* core : the engine and core system
* demo: primitive samples 
* tiles: Some pre-defined components (called tiles) including:
  * Autocomplete
  * HorizontalList
  * Input
  * Prompt
  * Table
  * VerticalList
